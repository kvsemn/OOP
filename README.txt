В директориях lab1 - lab9.1 представлены программы, написанные на C#, демонстрирующие понимание и умение 
работать с принцапами ООП.

Проект "Sipmle_Graph" - это пользовательское приложние по построению графов. В данной работе применяются
следующие технологии программирования: ООП, паттерн MVC(Model view controller), инструментальные средства 
по разработке desktop приложений.
